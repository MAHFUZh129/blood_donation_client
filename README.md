# BloodBridge – Save Lives with One Click ❤️

**Every 2 seconds someone needs blood.**  
BloodBridge is a modern, fast, and user-friendly blood donation platform that connects donors with patients in urgent need — instantly.

No more desperate Facebook posts.  
Just open the app, search by blood group & location, and find help within seconds.

**Live Demo** → [https://blood-donation-kappa-one.vercel.app/]

### Features
- Search donors by blood group & location
- Post emergency blood requests
- Register as a donor in under 1 minute
- View donor profiles & contact directly
- Fully responsive (mobile + desktop)
- Beautiful UI with red-gradient hero & daisyUI components

### Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS + daisyUI
- Deployed on Vercel
